# Android_Build_Shell

-------------------------------------------------------

**Folder Tree**

	├─UserVersionBuildShell
	└─SVN_Export_Compile_And_Error_Notify_By_RTX


`UserVersionBuildShell`
	用于自动化编译Android版本，在Shell下执行。
	Eng or Usr版本。
  
`SVN_Export_Compile_And_Error_Notify_By_RTX`
	SVN自动从SVN服务器获取代码并编译，并将编译结果通过腾讯通RTX发送
	到相关负责人；如果出现编译错误，将会通知到有提交的同事。
